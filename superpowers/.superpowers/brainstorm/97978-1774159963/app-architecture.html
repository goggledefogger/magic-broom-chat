Key Points

  • No custom backend server — React talks directly to Supabase
  • Auth — Supabase handles JWT sessions, signup/login flows
  • Real-time — Subscribe to new messages via Supabase Realtime (Postgres changes)
  • Presence — Supabase Presence API tracks online/offline/idle
  • Security — Row Level Security policies enforce permissions at the database level
  • Search — Postgres full-text search via tsvector, queried from the client
