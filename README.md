# Magic Broom Chat

A Zulip-like chat web app built four different ways to compare AI-assisted development frameworks.

This is a companion project for an educational course on AI-assisted software development. Each subfolder contains the same app concept — channels, threaded messages, user presence, and auth — built using a different methodology:

| Folder | Framework | Approach |
|--------|-----------|----------|
| `bmad/` | [BMAD Method](https://github.com/bmad-code-org/BMAD-METHOD) | Agent-based agile with specialized personas |
| `compound/` | [Compound Engineering](https://github.com/EveryInc/compound-engineering-plugin) | Plan-heavy cyclical workflow |
| `superpowers/` | [Superpowers](https://github.com/obra/superpowers) | TDD-first sequential phases |
| `vanilla/` | Native Claude Code | No framework — baseline comparison |

The `devlog/` folder tracks experiences, decisions, and observations across all four approaches.
